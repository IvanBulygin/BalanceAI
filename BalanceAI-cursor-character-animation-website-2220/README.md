# BalanceAI

AI-powered wellness website with an animated character companion that follows you through the page.

## Features

- **Animated Character** — A cute SVG mascot that floats alongside the page, reacting to each section with different animations (waving, thinking, excited, sparkling)
- **Eye Tracking** — The character's eyes follow your mouse cursor
- **Speech Bubbles** — Context-aware speech bubbles appear as you scroll through sections
- **Scroll Animations** — Elements fade in smoothly as they enter the viewport
- **Responsive Design** — Fully responsive layout that works on all devices
- **Modern UI** — Clean design with gradient accents, glass-morphism nav, and card-based layouts

## Getting Started

Open `index.html` in any modern browser. No build step required.
